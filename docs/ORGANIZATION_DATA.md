# 組織データ（入力用）

## 📋 入力ガイド
このドキュメントに組織図の情報を入力してください。入力が完了したら、このデータを基に実装を行います。

各ユーザーに対して以下を入力してください：
- **役職（ロール）**: 管理ダッシュボードへのアクセス権限を表します
  - `管理者`: 管理ダッシュボードにアクセス可能
  - `なし`: 管理ダッシュボードにアクセス不可（平社員）
- **担当業務**: ユーザーカードなどに表示されるバッジ（ステータス）のようなもの。視覚的な表示用
- **部署**: ユーザー管理ページでカテゴリ分けするために使用するもの
