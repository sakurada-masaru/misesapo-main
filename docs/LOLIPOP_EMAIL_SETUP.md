# ロリポップで従業員メールアドレス作成ガイド

## 📋 概要

ロリポップ（ロリポップ！レンタルサーバー）で`misesapo.app`ドメインのメールアドレスを作成する方法です。

### 基本プラン

- **ライトプラン**: 月額¥330（税込）〜
- **スタンダードプラン**: 月額¥550（税込）〜
- **プレミアムプラン**: 月額¥1,100（税込）〜

### メールアカウント

- **ライトプラン**: メールアカウント無制限
- **スタンダードプラン**: メールアカウント無制限
- **プレミアムプラン**: メールアカウント無制限

**重要**: ロリポップでは、どのプランでもメールアカウントは無制限に作成できます！

### 30個のメールアドレスを作成する場合

|-----|------|

**重要**: メールアカウントは無制限に作成できるため、30個でも追加費用は発生しません！

## 🚀 ロリポップでメールアドレスを作成する手順

### ステップ1: ロリポップのサーバーパネルにログイン

1. **ロリポップのサーバーパネルにアクセス**
   - https://lolipop.jp/
   - ユーザーIDとパスワードでログイン

### ステップ2: メールアカウントの作成

1. **左側のメニューから「メール設定／ロリポップ！webメーラー」**を選択
2. **「新規作成」ボタン**をクリック
3. **メールアカウント情報を入力**:
   - **メールアドレスの先頭部分**: `yamada.taro`（@より前の部分）
   - **パスワード**: 
     - 12文字以上32文字以下
     - 半角英字の大文字・小文字、半角数字、記号（「-」または「_」）をすべて組み合わせたもの
     - 例: `Yamada123!-`
4. **入力内容を確認**し、「作成」ボタンをクリック

**作成されるメールアドレス**: `yamada.taro@misesapo.app`

**注意**: 独自ドメイン（misesapo.app）を設定している場合、ライトプラン以上では無制限にメールアドレスを作成できます。

### ステップ3: メールクライアントの設定

#### 受信メールサーバー（IMAP）

- **サーバー名**: `imap.lolipop.jp`
- **ポート**: 993
- **セキュリティ**: SSL/TLS
- **ユーザー名**: `yamada.taro@misesapo.app`
- **パスワード**: 設定したパスワード

#### 送信メールサーバー（SMTP）

- **サーバー名**: `smtp.lolipop.jp`
- **ポート**: 465
- **セキュリティ**: SSL/TLS
- **認証**: 必要
- **ユーザー名**: `yamada.taro@misesapo.app`
- **パスワード**: 設定したパスワード

### ステップ4: Webメールでの確認

1. **ロリポップのサーバーパネル** → **「メール設定／ロリポップ！webメーラー」**
2. 作成したメールアドレスの右側にある「確認・変更」ボタンをクリック
3. 詳細情報を確認
4. Webメールでログインしてメールの送受信を確認

## 🔄 運用シナリオ

### シナリオ1: 新入社員のメールアドレス作成

1. ロリポップのユーザー専用ページにログイン
2. 「メール設定／ロリポップ！webメーラー」を選択
3. 「新規作成」ボタンをクリック
4. メールアドレスの先頭部分、パスワードを入力（12文字以上32文字以下、大文字・小文字・数字・記号を組み合わせ）
5. 作成完了
6. ユーザーにメールアドレスとパスワードを通知

### シナリオ2: 退職者のメールアドレス処理

1. ロリポップのユーザー専用ページにログイン
2. 「メール設定／ロリポップ！webメーラー」を選択
3. 該当のメールアカウントの「確認・変更」ボタンをクリック
4. 「削除」をクリック
5. または「転送設定」で別のメールアドレスに転送

### シナリオ3: メールアドレスの転送設定

1. ロリポップのユーザー専用ページにログイン
2. 「メール設定／ロリポップ！webメーラー」を選択
3. 該当のメールアカウントの「確認・変更」ボタンをクリック
4. 「転送設定」を編集
5. 転送先のメールアドレスを設定
