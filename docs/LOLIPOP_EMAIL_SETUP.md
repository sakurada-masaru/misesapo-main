## 🚀 ロリポップでメールアドレスを作成する手順

### ステップ1: ロリポップのサーバーパネルにログイン

1. **ロリポップのサーバーパネルにアクセス**
   - https://lolipop.jp/
   - ユーザーIDとパスワードでログイン

### ステップ2: メールアカウントの作成

1. **「メール」→「メールアカウント設定」**を選択
2. **「メールアカウント追加」**をクリック
3. **メールアカウント情報を入力**:
   - **メールアドレス名**: `yamada.taro`（@より前の部分）
   - **パスワード**: メールアドレスのパスワード
   - **メールボックス容量**: プランに応じて選択
   - **転送設定**: （オプション）別のメールアドレスに転送
4. **「確認画面へ進む」**をクリック
5. **「追加する」**をクリック

**作成されるメールアドレス**: `yamada.taro@misesapo.app`

### ステップ3: メールクライアントの設定

#### 受信サーバー（POP3/IMAP）

- **サーバー名**: `mail.misesapo.app` または `lolipop.jp`（プランによる）
- **ポート**: 
  - POP3: 110（通常）/ 995（SSL）
  - IMAP: 143（通常）/ 993（SSL）
- **ユーザー名**: `yamada.taro@misesapo.app`
- **パスワード**: 設定したパスワード

#### 送信サーバー（SMTP）

- **サーバー名**: `mail.misesapo.app` または `lolipop.jp`（プランによる）
- **ポート**: 587（STARTTLS）または 465（SSL）
- **認証**: 必要
- **ユーザー名**: `yamada.taro@misesapo.app`
- **パスワード**: 設定したパスワード

### ステップ4: Webメールでの確認

1. **ロリポップのサーバーパネル** → **「メール」→「Webメール」**
2. メールアドレスとパスワードでログイン
3. メールの送受信を確認

## 🔄 運用シナリオ

### シナリオ1: 新入社員のメールアドレス作成

1. ロリポップのサーバーパネルにログイン
2. 「メール」→「メールアカウント設定」→「メールアカウント追加」
3. メールアドレス名、パスワードを入力
4. 作成完了
5. ユーザーにメールアドレスとパスワードを通知

1. ロリポップのサーバーパネルにログイン

4. 「削除」をクリック
5. または「転送設定」で別のメールアドレスに転送

### シナリオ3: メールアドレスの転送設定

1. ロリポップのサーバーパネルにログイン
2. 「メール」→「メールアカウント設定」
3. 該当のメールアカウントを選択
4. 「転送設定」を編集
5. 転送先のメールアドレスを設定
