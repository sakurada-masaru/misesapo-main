# Xサーバーで30個のメールアドレスを作成する場合のコスト分析

## 📊 結論

**Xサーバーで30個のメールアドレスを作成するのが最もコスト効率が良いです！**

- **月額料金**: ¥990/月（税込）〜

- **メールアカウント**: **無制限**（追加費用なし）

| 項目 | 金額 |
|-----|------|
| **初期費用** | ¥3,300（初回のみ） |
| **月額料金** | ¥990/月 |
| **メールアドレス30個** | **¥0（追加費用なし）** |
| **年間コスト** | ¥11,880（月額¥990 × 12ヶ月） |
| **初年度合計** | ¥15,180（初期費用 + 年間コスト） |

**重要**: メールアカウントは無制限に作成できるため、30個でも追加費用は発生しません！

## 📝 実装手順

### ステップ1: Xサーバーのプランを確認

1. Xサーバーのサーバーパネルにログイン
2. 現在のプランを確認
3. メールアカウントの作成可能数を確認（通常は無制限）

### ステップ2: メールアドレスの作成

1. サーバーパネル → 「メール」→「メールアカウント設定」
2. 「メールアカウント追加」をクリック
3. 各従業員のメールアドレスを作成
   - 例: `yamada.taro@misesapo.app`
   - 例: `suzuki.hanako@misesapo.app`
   - ...（30個まで）

### ステップ3: AWS Cognitoに登録

1. 各従業員をAWS Cognitoに登録
2. メールアドレス + パスワードで認証
3. DynamoDBの`workers`テーブルに保存
