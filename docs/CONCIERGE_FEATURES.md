# コンシェルジュページ機能要件

## 基本方針
- **SPメイン（スマートフォン優先）**: 現場での使用を想定
- **タッチ操作最適化**: ワンタップでアクション可能
- **重要な情報を優先表示**: 緊急性の高い情報を上部に配置
- **オフライン対応**: 基本情報はキャッシュ可能に

---

## 必須機能

### 1. 顧客一覧
- **説明**: 全顧客の一覧表示（これは顧客一覧ページへのリンクでOK）
- **UI**: 大きなボタンまたはカード形式で「顧客一覧を見る」を表示
- **リンク先**: `/sales/clients.html`

### 2. 本日のスケジュール
- **説明**: 本日の訪問予定を時系列で表示
- **表示内容**:
  - 訪問時間
  - 顧客名・店舗名
  - 訪問目的（見積もり、商談、定期訪問など）
  - 場所（住所）
  - 地図へのリンク（Google Mapsなど）
- **UI**: カード形式、時間順にソート
- **アクション**: 
  - 顧客詳細へのリンク
  - 地図アプリで開く
  - 電話・メールへの直接リンク

### 3. 緊急対応が必要な顧客
- **説明**: 対応が必要な顧客を優先表示
- **対象**:
  - 見積もり承認待ち（3日以上経過）
  - 発注承認待ち
  - 新規問い合わせ（未対応）
  - 清掃トラブル報告
- **UI**: 警告色（赤・オレンジ）で強調表示
- **アクション**: 各顧客の対応ページへ直接リンク

### 4. 見積もり状況
- **説明**: 見積もりの状況を簡易表示
- **表示内容**:
  - 承認待ち見積もり数
  - 送付済み見積もり数（未承認）
  - 今週の見積もり作成数
- **UI**: 統計カード形式
- **リンク**: `/sales/estimates.html`（フィルター付き）

### 5. 新規問い合わせ・通知
- **説明**: 未対応の新規問い合わせを表示
- **表示内容**:
  - 問い合わせ日時
  - 会社名
  - 問い合わせ内容の要約
- **UI**: 通知バッジ付きカード
- **アクション**: 対応ボタン（新規顧客登録ページへ）

### 6. 売上サマリー
- **説明**: 今月・今週の売上を表示
- **表示内容**:
  - 今月の売上
  - 今週の売上
  - 目標達成率
  - 前月比
- **UI**: 統計カード形式
- **リンク**: `/sales/analytics.html`（詳細分析）

### 7. クイックアクション
- **説明**: よく使う機能へのショートカット
- **項目**:
  - 新規顧客登録
  - 見積もり作成
  - 発注管理
  - レポート確認
  - スケジュール管理
- **UI**: アイコンボタン形式（タッチしやすいサイズ）

### 8. 最近訪問した顧客
- **説明**: 最近訪問した顧客を表示（5件程度）
- **表示内容**:
  - 顧客名・店舗名
  - 訪問日時
  - 訪問目的
- **UI**: カード形式
- **アクション**: 顧客詳細へのリンク

### 9. 次回訪問予定
- **説明**: 今後の訪問予定を表示（1週間分）
- **表示内容**:
  - 訪問日時
  - 顧客名・店舗名
  - 訪問目的
- **UI**: カレンダー形式またはリスト形式
- **アクション**: スケジュール詳細ページへ

## 推奨機能（現場で使い勝手が良い）

### 10. 現在位置から近い顧客
- **説明**: GPS連携で現在位置から近い顧客を表示
- **表示内容**:
  - 距離（km）
  - 顧客名・店舗名
  - 到着予想時間
  - 地図へのリンク
- **UI**: 距離順にソート、地図アイコン付き
- **アクション**: 
  - ナビゲーションアプリで開く
  - 顧客詳細へのリンク

### 11. 電話・メールへの直接リンク
- **説明**: 顧客情報に電話・メールへの直接リンクを配置
- **UI**: 
  - 電話アイコン → `tel:`リンク
  - メールアイコン → `mailto:`リンク
- **配置**: 顧客カード、顧客詳細ページ

### 12. 地図表示・ナビゲーション
- **説明**: 顧客の住所を地図アプリで開く
- **UI**: 
  - 地図アイコン
  - Google Maps、Apple Mapsで開く
- **配置**: 顧客カード、スケジュールカード

### 13. メモ・履歴の表示
- **説明**: 顧客に関する最新メモ・訪問履歴を表示
- **表示内容**:
  - 最新のメモ（3件程度）
  - 訪問履歴（日付、目的）
- **UI**: コンパクトなリスト形式
- **アクション**: 詳細ページで全履歴を確認

### 14. よく使う顧客のショートカット
- **説明**: よく訪問する顧客をピン留めして表示
- **UI**: 固定セクション、カード形式
- **機能**: ピン留め/解除機能

### 15. 営業活動記録（訪問履歴）
- **説明**: 訪問履歴を記録・表示
- **表示内容**:
  - 訪問日時
  - 訪問先
  - 訪問目的
  - 訪問結果（商談、見積もり提出など）
- **UI**: タイムライン形式
- **アクション**: 訪問記録の追加・編集

### 16. 見積もり・発注のクイック確認
- **説明**: 承認待ちの見積もり・発注を一覧表示
- **表示内容**:
  - 見積もり番号・発注番号
  - 顧客名
  - 金額
  - ステータス
- **UI**: カード形式、ステータスバッジ付き
- **アクション**: 詳細ページへリンク

### 17. 売上目標・達成率
- **説明**: 個人の売上目標と達成率を表示
- **表示内容**:
  - 今月の売上
  - 目標金額
  - 達成率（%）
  - 残り目標
- **UI**: プログレスバー形式
- **リンク**: 詳細分析ページへ

### 18. 通知・アラート
- **説明**: 重要な通知を表示
- **対象**:
  - 見積もり承認待ち（期限切れ間近）
  - 発注承認待ち
  - 新規問い合わせ
  - 清掃トラブル報告
- **UI**: 通知バッジ、トースト通知
- **アクション**: 各通知の詳細ページへ

### 19. オフライン機能
- **説明**: オフラインでも基本情報を確認可能
- **対象**:
  - 顧客基本情報
  - 最近訪問した顧客
  - スケジュール（1週間分）
- **UI**: オフライン時はバナー表示
- **機能**: Service Workerでキャッシュ

### 20. 検索機能
- **説明**: 顧客・見積もり・発注を横断検索
- **検索対象**:
  - 顧客名
  - 店舗名
  - 見積もり番号
  - 発注番号
- **UI**: 検索バーをヘッダーに配置
- **機能**: リアルタイム検索、検索履歴

## UI/UX 配慮事項

### タッチ操作最適化
- **ボタンサイズ**: 最小44x44px（タップしやすいサイズ）
- **間隔**: ボタン間の間隔を適切に（誤タップ防止）
- **スワイプ操作**: カードのスワイプで削除・アーカイブ

### 情報の優先順位
1. **緊急対応が必要な顧客**（最上部）
2. **本日のスケジュール**
3. **新規問い合わせ**
4. **見積もり状況**
5. **その他の情報**

### 視覚的な階層
- **重要情報**: 大きなカード、警告色
- **通常情報**: 標準カード、通常色
- **参考情報**: 小さなカード、グレー系

### レスポンシブ対応
- **SP**: 1列レイアウト、大きなタップ領域
- **PC**: 2列レイアウト、マウス操作対応

### パフォーマンス
- **読み込み速度**: 重要情報を優先読み込み
- **画像最適化**: 遅延読み込み、WebP形式
- **キャッシュ**: Service Workerでキャッシュ
