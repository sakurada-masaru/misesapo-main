レポート編集のヘルプ

ボタン操作の概要

1. 清掃項目を追加

清掃項目を選択または自由入力で追加します。タイプ（例: 床置き型）や個数（例: 2個）を指定できます。

  • ドロップダウンから既存の清掃項目を選択
  • 「その他（自由入力）」を選択すると、自由に項目名を入力できます
  • 未入力の清掃項目は保存時に除外されます
2. 画像を追加

画像セクションを追加します。2つのパターンから選択できます：

  • 作業前・作業後：清掃作業の前後の写真を追加
  • 設置前・設置後：設備の設置前後の写真を追加

各画像セクションでは、以下の方法で画像を追加できます：

  • ファイルから：PCやスマートフォンから画像を直接アップロード
  • メディアから：メディアライブラリから画像を選択
  • 画像倉庫から：過去にアップロードした画像を日付やカテゴリで検索して選択
3. コメントを追加

自由形式のコメントを追加します。特記事項や補足情報を記載できます。

  • 空のコメントセクションは保存時に除外されます
4. 作業内容を追加

作業内容の詳細を追加します。実施した作業の詳細を記載できます。

  • 空の作業内容セクションは保存時に除外されます

注意事項

  • 店舗と清掃日は必須項目です
  • 少なくとも1つの項目（清掃項目、画像、コメント、作業内容のいずれか）を追加する必要があります
  • 各セクションは右上の「削除」ボタンで個別に削除できます
  • 画像は最大800px幅に自動リサイズされます
  • 画像倉庫から選択した画像は、日付とカテゴリ（作業前/作業後）で検索できます
  • レポートを保存すると、画像は自動的にS3にアップロードされます

レポートが生成される仕組み

1. レポート作成の流れ
  1. 基本情報（店舗、清掃日、開始時刻、終了時刻、担当者）を入力
  2. 各種セクション（清掃項目、画像、コメント、作業内容）を追加
  3. 「レポートを作成」ボタンをクリック
  4. データがバリデーションされ、APIに送信されます
2. データの保存先
  • テキストデータ：DynamoDBに保存されます
  • 画像データ：AWS S3に保存されます（日付ベースのフォルダ構造）
  • 画像のメタデータ（URL、日付、カテゴリ）もDynamoDBに保存されます
3. レポートの共有
  • レポート一覧から「URL発行」ボタンをクリック
  • 生成されたURLを顧客に送信すると、顧客はレポートを直接閲覧できます
  • 共有URLは認証不要でアクセス可能です
4. 画像の管理
  • アップロードされた画像は「画像倉庫」に保存されます
  • 画像倉庫では、日付とカテゴリ（作業前/作業後）で検索できます
  • 過去のレポートで使用した画像を再利用できます
  • 画像は日付ベースのフォルダ構造で整理されます（例: before/2025/12/01/ID0001.jpg）
